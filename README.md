# 📦 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/smart-campus-hub.git
cd smart-campus-hub
```

---

## 2️⃣ Setup Backend

```bash
cd backend
npm install
```

### Configure Environment Variables

Create `.env`:

```env
PORT=5000
JWT_SECRET=your_super_secure_secret
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=smart_campus
CLIENT_ORIGIN=http://localhost:5173
```

---

## 3️⃣ Setup Frontend

```bash
cd frontend
npm install
```

Create `.env`:

```env
VITE_API_BASE=http://localhost:5000
```

---

## 4️⃣ Setup Database

```bash
mysql -u root -p
```

```sql
CREATE DATABASE smart_campus;
```

Import schema:

```bash
mysql -u root -p smart_campus < backend/sql/smart_campus.sql
```

Run migrations:

```bash
mysql -u root -p smart_campus < backend/sql/migrations/002_results.sql
mysql -u root -p smart_campus < backend/sql/migrations/003_ai_analytics_cache.sql
```

---

## 5️⃣ Run Development Servers

### Backend

```bash
cd backend
npm run dev
```

### Frontend

```bash
cd frontend
npm run dev
```
